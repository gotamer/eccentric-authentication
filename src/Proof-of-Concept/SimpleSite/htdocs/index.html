Simple site for Eccentric CA

Proof of concept

This is the index page. It shows the content you can read without logging in. For example, the most recent items on a blog.

To write a comment, or your own blog, you need to be logged in. This link takes you there.
If you have registered at the Eccentric CA and loaded the certificate (pseudonym/identity) into you browser, please select it and you're ready to write.
If you don't have an registered identity, it will show you the where to sign up.

Sign up

You can sign up for (more) identities at https://127.0.0.1:7444/register. All you need is a nickname that you decide and a private key. It tells you how to do that.
